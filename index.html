Offline voice recognition (no API key)

Download a small speech model once (~80 MB, use Wi-Fi) and transcription runs entirely on your phone — works with no internet and nothing leaves the device.

Household members (optional)

Add names, comma-separated, and tasks mentioning them are tagged automatically — say "remind Lulu to return the library books" and it lands under her tag, so each person can filter to their own tasks on the Today view.

Family sync (beta)

Share one task list across the household — new tasks and ticks appear on everyone's phone in real time, and it works offline (changes sync when you're back online). Ideas and voice transcripts stay private on each device; only tasks are shared.

First device? Create the household (one-time, ~3 min)

1. Go to console.firebase.google.com → Add project (any name, Analytics off).
2. Build → Firestore Database → Create database (production mode).
3. In the Rules tab, paste the rules from the README and Publish.
4. Project settings (⚙) → Your apps → Web (</>) → Register app → copy the firebaseConfig snippet.
5. Paste it below and tap Create.

Daily digest notification

Get a notification once or twice a day (timing chosen by your phone) with your open tasks and what the family added or ticked — even while the app is closed. Needs the app installed via Chrome, and Family sync active.

Smart summaries — free (Gemini)

Paste a free Google Gemini API key and summaries, priorities, and tags are generated by Gemini — much smarter than the built-in mode, at no cost. Get a key in under a minute at aistudio.google.com/apikey (no credit card needed).

Smart summaries — Claude (optional)

Alternatively, paste an Anthropic API key for Claude-quality summaries (paid, a fraction of a cent per capture). If both keys are set, Gemini is used first and Claude is the backup.

Daily reset

Each day, finished tasks are archived and unfinished ones carry over to today automatically.

Your data

Everything is stored only on this device (localStorage). Nothing is uploaded anywhere unless you enable Gemini or Claude summaries — then only the idea text is sent, using your own key. Use backups to move your data to another device or browser.
